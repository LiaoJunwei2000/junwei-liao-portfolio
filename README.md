# Junwei Liao - Personal Portfolio

A clean and professional personal portfolio website showcasing my experience, education, and achievements.

## Design Philosophy

This website follows the **Academic Minimalism** design approach, inspired by modern academic portfolios:

- Clean information hierarchy with ample whitespace
- Deep blue accent color for emphasis
- Inter font family for modern readability
- Single column layout optimized for reading
- Timeline visualization for career progression

## Features

- **Responsive Design**: Works seamlessly on desktop, tablet, and mobile devices
- **Modern Tech Stack**: Built with React 19, Tailwind CSS 4, and Vite
- **Fast Loading**: Optimized for performance
- **Accessible**: Follows web accessibility best practices

## Sections

- **Hero**: Introduction with social links
- **About**: Professional summary
- **Experience**: Career timeline with detailed descriptions
- **Education**: Academic background
- **Certifications**: Professional licenses and certifications
- **Honors & Awards**: Recognition and achievements

## Tech Stack

- **Framework**: React 19
- **Styling**: Tailwind CSS 4
- **UI Components**: shadcn/ui
- **Build Tool**: Vite
- **Icons**: Lucide React

## Development

```bash
# Install dependencies
pnpm install

# Start development server
pnpm dev

# Build for production
pnpm build
```

## Deployment

This website can be deployed to any static hosting service:

- Vercel
- Netlify
- GitHub Pages
- Cloudflare Pages
